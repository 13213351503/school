  
react特点:虚拟DOM(数据驱动UI)

开发框架:
1. 数据(state)
2. 模板(jsx语法)
3. 数据 + 模板 => 原始的真实DOM节点
4. 数据改变(state change)
5. 新的数据 + 模板 => 生成新的真实的DOM节点
6. 原始的真实DOM节点 => 新的DOM节点



react框架开发:
1. 数据(state)
2. 模板(jsx语法)
3. 数据 + 模板 => 原始的真实DOM节点
4. 由原始的真实DOM节点会生成原始的虚拟的DOM节点(一个用来描述dom节点的对象)
5. 数据改变(state change)
6. 新的数据 + 模板 => 新的虚拟的DOM节点
7. 原始的虚拟的DOM节点和新的虚拟的DOM节点进行比较(diff算法)
8. 更新原始的DOM节点(只更新有变化的部分)
